🔌 Wiring Connections

NodeMCU → MAX7219 LED Matrix

NodeMCU PinMAX7219 PinNotes
D7 (GPIO13)→DINData In
D5 (GPIO14)→CLKClock
D8 (GPIO15)→CSChip Select
—→VCCConnect to external 5V (not NodeMCU 3.3V)
GND→GNDCommon ground

Arduino Uno → Soil Sensor & Relay

Arduino PinComponentNotes
A0→Soil Sensor (AO)Analog moisture reading
D4→Relay INControls pump ON/OFF
5V→Soil VCC + Relay VCCPower for both
GND→Soil GND + Relay GNDCommon ground

NodeMCU ↔ Arduino Serial Communication

NodeMCU PinArduino PinNotes
D6 (GPIO12) TX→D2 (SoftSerial RX)NodeMCU sends commands
D1 (GPIO5) RX←D3 (SoftSerial TX)Via voltage divider (5V→3.3V)
GND↔GNDMust share common ground
⚠️ Important: NodeMCU is 3.3V logic. Use a voltage divider (two resistors: 1kΩ + 2kΩ) on the Arduino TX → NodeMCU RX line to avoid damage!

Pump Wiring (via Relay)

ConnectionToNotes
Relay COM→Pump (+) wireCommon terminal
Relay NO→External 5V (+)Normally Open — pump runs when relay activates
Pump (−)→External 5V GNDComplete the circuit
💡 Tip: Use a separate 5V power supply for the pump. Drawing too much current from Arduino's 5V pin can cause resets.

💻 Software Setup

Step 1: Install Arduino IDE

  1. 1Download Arduino IDE from arduino.cc/en/software
  2. 2Install ESP8266 board support: Go to File → Preferences, add this URL to Board Manager URLs:
    http://arduino.esp8266.com/stable/package_esp8266com_index.json
  3. 3Go to Tools → Board → Board Manager, search "ESP8266" and install

Step 2: Install Required Libraries

  1. 1MD_Parola — LED matrix text scrolling
  2. 2MD_MAX72xx — MAX7219 hardware driver
  3. 3ArduinoJson — Parse weather API response
  4. 4ESP8266WiFi — Built-in with ESP8266 board package
  5. 5ESP8266HTTPClient — Built-in HTTP requests
  6. 6SoftwareSerial — Built-in with Arduino IDE
💡 Install libraries via Sketch → Include Library → Manage Libraries and search by name.

🧪 Testing & Calibration

  1. 1Test WiFi: Open Serial Monitor (115200 baud) on NodeMCU. You should see "WiFi Connected" and your IP address.
  2. 2Test LED Matrix: "SMARTPLANT STARTING..." should scroll across the display on boot.
  3. 3Test Soil Sensor: Open Serial Monitor on Arduino (9600 baud). Check analog readings: Dry soil ≈ 700-1023, Wet soil ≈ 200-400.
  4. 4Calibrate Thresholds: Adjust DRY_THRESHOLD and WET_THRESHOLD in the Arduino code based on your sensor readings.
  5. 5Test Pump: Press the soil sensor into dry soil (or leave it in air). The relay should click and pump should start.
  6. 6Test Serial Comms: Both Serial Monitors should show data exchange between NodeMCU and Arduino.
💡 Calibration Tip: Dip the soil sensor in a glass of water to find your "100% wet" value, then let it dry completely to find "0% wet" value. Use these as your min/max.

ProblemPossible CauseFix
LED matrix blankWrong wiring or 3.3V powerUse external 5V for MAX7219, check DIN/CLK/CS pins
WiFi won't connectWrong credentialsDouble-check SSID and password, ensure 2.4GHz network
Soil readings stuck at 1023Sensor not connectedCheck A0 wiring, ensure VCC is powered
Pump won't turn onRelay wiring or powerTest relay with LED first, check NO vs NC terminals
NodeMCU resets randomlyPower draw too highUse separate 5V supply for pump and LED matrix
Garbled serial dataBaud rate mismatchBoth devices must use same baud rate (9600)
Weather data "N/A"API request failingCheck WiFi, test URL in browser, verify coordinates
